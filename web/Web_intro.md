# HTML
: 웹 페이지의 의미와 구조를 정의하는 언어  
- Hypertext: 웹 페이지를 다른 페이지로 연결하는 링크 
- Markup Language: 태그 등을 이용하여 문서나 데이터의 구조를 명시하는 언어
